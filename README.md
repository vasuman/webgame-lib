A small general purpose library for webgames written in ES6.

# Installation

```
$ git clone https://github.com/vasuman/webgame-lib
$ cd webgame-lib
$ npm install
$ npm link
```

# Usage

Either import the whole package,

```
require('webgame-lib')
```

Or use specific modules,

```
require('webgame-lib/lib/<module-name>')
```
